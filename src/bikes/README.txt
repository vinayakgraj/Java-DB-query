README 

please use the following format to enter the data

-type .. -material .. -color .. -gear .. -height .. -wheelbase ..

the arguments can be set in any order , but the last argument always has to be gear/height/wheelbase without a value for it .

example :-  -type street_bike -wheelbase 36  -material steel -color red -height 4 -gear

				the above example will fetch with the bikes the given attributes and sort them according to the gear attribute .

The code has been tested in Eclipse IDE . 

The database has been provided from which the code can be tested .
